## Post Comment App For Learn Microservice

### Instalation Client
1. Open Client Folder :
```bash
    cd client
```

2. run npm install :
```bash
    npm install
```

3. running app : 
```bash
    npm run dev
```

### Instalation Posts
1. Open posts Folder :
```bash
    cd posts
```

2. run npm install :
```bash

```

3. running app :
